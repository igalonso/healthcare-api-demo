Healthcare API Demo

Welcome to the Healthcare API demo from Google Cloud Platform

This is a short demo to demonstrate 2 main features: Consent and image deidentification. This demo takes the point of view of 3 roles: researcher, doctor and the most important, the patient! You can find all the code use for this demo here:

Github repo!

Researcher Role
